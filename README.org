* Como añadir una canción:
1. Crear un archivo .qmd
2. Editar el texto
3. Añadir el nombre del archivo bajo la lista "chapters" en _quarto.yml
   
Para un ejemplo, ver el archivo alicantina.qmd
* Ver como quedará la canción
1. Descargar el programa "quarto"
2. En un terminal, dentro del fichero con los archivos, ejecutar el comando siguiente:
#+begin_src bash
quarto preview .
#+end_src

* Como publicar al sitio web
Basta crear un pull request con la canción
